Digital Hour Program ⏰
Description

This is a simple Python program that allows the user to enter a time and display it in a digital clock format (HH:MM:SS).
The program ensures that the entered hours, minutes, and seconds are within valid ranges.

Features

User-friendly input with error checking.

Ensures hours are between 0 and 23.

Ensures minutes and seconds are between 0 and 59.

Prints the time in a clean digital format with two digits for each value (e.g., 09:05:03).

How to Use

Run the program in a Python environment (Python 3 recommended).

Enter the hour (0–23) when prompted.

Enter the minute (0–59) when prompted.

Enter the second (0–59) when prompted.

The program will display the digital time in HH:MM:SS format.

Example:

Please enter the hour: 9
Please enter the minute: 5
Please enter the second: 3
⏰ The digital time is: 09:05:03

Error Handling

If the user enters a value outside the valid range, the program will ask again.

If the user enters a non-numeric value, the program will prompt to enter a number.

Requirements

Python 3.x

No external libraries required.
